# 11 Avatar  - All in One Image Conversion & Optimizer Tool


11 Avatar is a powerful, client-side media conversion tool that allows users to convert, optimize, and enhance multiple images simultaneously. With support for 10+ image formats, advanced batch processing capabilities, and a privacy-focused approach (all processing happens in the browser), 11 Avatar is the perfect solution for photographers, designers, and content creators.

## Key Features

- **Batch Processing**: Convert hundreds of images at once
- **Advanced Batch Processing**: Automate multiple operations across image sets
- **Wide Format Support**: JPG, PNG, WEBP, GIF, SVG, HEIC, and RAW formats
- **Resizing Options**: Resize images in bulk with precision
- **Rotation**: Rotate images left, right, or flip as needed
- **Quality Adjustment**: Control image compression and quality level
- **Batch Renaming**: Rename files in bulk using custom patterns
- **Image Enhancement & Filters**: Apply sharpness, blur, contrast, grayscale, and more
- **Watermarking**: Add logos or text watermarks to protect your work
- **Compression Level Control**: Choose how much to compress images for best balance
- **Color Space Conversion**: Switch between color profiles like RGB and CMYK
- **DPI Resolution Adjustment**: Customize dots per inch for printing needs
- **EXIF Data Preservation**: Keep or strip metadata as needed
- **Web Optimization**: Optimize images for faster website loading
- **Real-Time Stats**: Get instant feedback on processed image counts and sizes
- **Dark Mode Toggle**: Switch between light and dark themes for better comfort
- **Toast Notifications**: Get smooth, unobtrusive action updates
- **Privacy Focused**: All processing happens locally in your browser – no uploads
- **Responsive Design**: Fully functional across desktop and mobile devices


## Installation

1. Download the package from CodeCanyon
2. Unzip the files to your web server
3. Open `index.html` in a browser
4. No server-side requirements - works with any standard web hosting

## Requirements

- Modern browser (Chrome, Firefox, Edge, Safari 12+)
- JavaScript enabled
- For HEIC/RAW support: Chrome or Edge recommended
